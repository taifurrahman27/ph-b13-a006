## Project Name: 🚀 Digitools



## 📌 Project Overview

**Digitools** is a modern web application that presents a curated selection of premium digital tools aimed at boosting productivity and fostering creativity. It features an intuitive interface, seamless interactions, and an engaging user experience for discovering powerful tools in one convenient place.



## 🛠️ Technologies Used

* ⚛️ React

* 🎨 Tailwind CSS

* 🌼 DaisyUI

* ⚡ Vite



## ✨ Features

* 🔥 **Beautiful UI Design** — Clean, modern, and responsive interface using Tailwind & DaisyUI

* ⚡ **Fast Performance** — Built with Vite for lightning-fast development and load times

* 🛒 **Interactive Tabs & Cart System** — Easily switch between products and cart with dynamic UI
